Grupa CE - git config --global user.email ""
Polecenie służy do ustanowienia lub zmiany adresu e-mail powiązanego z Gitem.

- git diff --staged
Polecenie służy do szczegółowego wyświetlenia zmian w pliku

- git checkout -b "branch"
W ramach naszego repozytorium, to polecenie tworzy jego gałąź o nazwie podanej w cudzysłowiu, czyli branch, i się na nią przełącza.

- git log --graph --all
Polecenie wyświetlające historię naszego repozytorium w postacji graficznej, czyli w tym wypadku jako graf.

- git branch -d "branch"
Polecenie to usuwa gałąź o nazwie "branch" w naszym repozytorium.

- git fetch "remote" "branch"
Polecenie służy do pobrania na swoje repozytorium zawartości repozytoriów lub gałęzi o nazwach "remote" i "branch".
